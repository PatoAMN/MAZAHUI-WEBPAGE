# MAZAHUI - Sitio Web Moderno

## Descripción

Este es un sitio web moderno y minimalista para MAZAHUI, una asociación civil sin fines de lucro dedicada a preservar la cultura mazahua y promover la igualdad de oportunidades.

## Características

- **Diseño Moderno**: Interfaz limpia y minimalista con gradientes y sombras sutiles
- **Responsive**: Completamente adaptable a dispositivos móviles y de escritorio
- **Navegación Suave**: Scroll suave entre secciones y navegación fija
- **Animaciones**: Efectos de entrada y hover para mejorar la experiencia del usuario
- **Formulario de Contacto**: Sistema de contacto funcional con validación
- **Optimizado para SEO**: Estructura semántica y meta tags apropiados

## Estructura del Sitio

### Secciones Principales

1. **Hero**: Página de bienvenida con mensaje principal y botones de acción
2. **Conócenos**: Historia y misión de la organización
3. **Programas**: Cuatro programas principales con descripciones detalladas
4. **Resultados**: Estadísticas y logros de 20 años de trabajo
5. **Tienda**: Información sobre productos artesanales y beneficios
6. **Apóyanos**: Formas de contribuir y apoyar la causa
7. **Testimonios**: Experiencias de beneficiarias
8. **Contacto**: Información de contacto y formulario

### Navegación

- Menú desplegable para la sección de Programas
- Navegación fija con efecto de transparencia
- Menú hamburguesa para dispositivos móviles
- Enlaces internos con scroll suave

## Tecnologías Utilizadas

- **HTML5**: Estructura semántica moderna
- **CSS3**: Estilos avanzados con Grid, Flexbox y animaciones
- **JavaScript ES6+**: Funcionalidad interactiva y animaciones
- **Font Awesome**: Iconografía moderna
- **Google Fonts**: Tipografía Inter para mejor legibilidad

## Instalación y Uso

### Requisitos

- Navegador web moderno (Chrome, Firefox, Safari, Edge)
- Servidor web local o hosting

### Instalación

1. Descarga todos los archivos del proyecto
2. Coloca los archivos en tu servidor web o directorio local
3. Abre `index.html` en tu navegador

### Estructura de Archivos

```
MAZAHUI/
├── index.html          # Página principal
├── styles.css          # Estilos CSS
├── script.js           # Funcionalidad JavaScript
└── README.md           # Este archivo
```

## Personalización

### Colores

Los colores principales se pueden modificar en `styles.css`:

```css
:root {
    --primary-color: #667eea;
    --secondary-color: #764ba2;
    --text-color: #333;
    --text-light: #666;
    --background-light: #f8f9fa;
}
```

### Contenido

- **Texto**: Modifica el contenido en `index.html`
- **Imágenes**: Reemplaza los placeholders con imágenes reales
- **Enlaces**: Actualiza los enlaces de redes sociales y contacto

### Funcionalidad

- **Formulario**: El formulario de contacto está configurado para mostrar notificaciones
- **Animaciones**: Las animaciones se pueden ajustar en `script.js`
- **Navegación**: El menú móvil se activa automáticamente en pantallas pequeñas

## Características Responsivas

- **Desktop**: Layout de dos columnas con navegación completa
- **Tablet**: Adaptación automática a una columna
- **Mobile**: Menú hamburguesa y layout optimizado para pantallas pequeñas

## Optimizaciones

- **Performance**: CSS y JavaScript optimizados
- **Accesibilidad**: Estructura semántica y navegación por teclado
- **SEO**: Meta tags y estructura HTML apropiada
- **Cross-browser**: Compatible con navegadores modernos
